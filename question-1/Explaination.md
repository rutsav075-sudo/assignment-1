# Command Explanations - Question 1

- `whoami`: I executed this command to display the current logged-in username, verifying the user account operating in the terminal.
- `groups`: I ran this to list all the user groups my account belongs to, establishing the baseline permissions assigned to my user.
- `echo $SHELL`: I printed this environment variable to determine which command-line interpreter I am using, confirming it is set to `/bin/zsh`.
- `pwd`: I used the print working directory command to find my exact location in the file system and ensure I am in the correct workspace.
- `ls -la`: I executed this to list all files and directories in my current workspace, including hidden files, to view the contents and their permissions.
- `ping -c 4 google.com`: I ran a ping command with a count of 4 packets directed at google.com to verify that the machine has successful external network connectivity.
